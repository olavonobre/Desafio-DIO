Codigo-fonte e Pré-formatação

O comando document.getElementById('teste') é escrito em linguagem JavaScript, para ficar neste formato basta usar a tag < code >

usando a tag < code > o codigo nao fica como queríamos

num = int(imput('Digite um número')) if num % 2 == 0: print(f'O número {num} é PAR') else: print(f'O número {num} é ÍMPAR') print('Fim do Programa')

para ficar no formato de código, como conhecemos..temos que usar a tag < pre > envelopando com o CTRL+SHIFT+P tudo que esta dentro da tag < code >

        
            num = int(imput('Digite um número'))
            if num % 2 == 0:
                print(f'O número {num} é PAR')
            else:
                print(f'O número {num} é ÍMPAR')
            print('Fim do Programa')

E assim fica da forma que conhecemos o código
